이미지(image)

image attribute

  • src attribute : 이미지 파일 경로
  • alt attribute : 이미지 파일이 없을 경우 표시되는 문장
  • width attribute : 이미지의 너비 (CSS에서 지정하는 것이 일반적)
  • height attribute : 이미지의 높이 (CSS에서 지정하는 것이 일반적)

미디어(media)

오디오(audio)

  • audio 태그는 HTML5에서 새롭게 추가된 태그이다.
  • IE8 이하에서는 사용할 수 없다.
  • attribute로 src, preload, autoplay, loop, controls 등이 있다.
  • 웹 브라우저별로 지원하는 음악 파일 형식이 다르다.

  • source 태그를 사용하여 파일 형식의 차이 문제를 해결할 수 있다.
  • type 어트리뷰트는 생략 가능하다.

비디오(video)

  • video 태그는 HTML5에서 새롭게 추가된 태그이다.
  • IE8 이하에서는 사용할 수 없다.
  • src, poster, preload, autoplay, loop, controls, width, height 등의 attribute를 갖는다.
  • audio 태그와 마찬가지로 파일 형식의 차이 문제가 발생할 수 있다.
  • source 태그를 사용하여 형식 차이 문제를 해결한다. type 어트리뷰트는 생략 가능하다.
