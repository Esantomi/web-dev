제목 1: 글자 형태 (Text Formatting) 태그

b로 볼드 설정

p style="font-weight: bold;"로 볼드 설정

strong으로 볼드 설정

제목 2: small tag 사용

i로 이탤릭 설정

p style="font-style: italic;"로 이탤릭 설정

em으로 이탤릭 설정 (emphasized)

제목 3: mark tag 사용

del tag를 쓰면 이렇게 텍스트 취소선이 가능하다.

이건 ins tag인데 insert란 뜻이지만 밑줄 같다.

sub tag는 subscripted, sup tag는 supercripted를 뜻한다.

제목 4: 본문 태그 (1)

p 태그는 단락(paragraph)을 지정한다.

이렇게 p 태그가 연속으로 오면 단락이 바뀐다.

br tag는 개행을 지정한다.
빈 요소(empty element)이므로 종료 태그가 없다.

띄어쓰기를 길게 하고 싶으면    를 쓴다.(& nbsp;)

제목 5 : 본문 태그 (2)
            pre tag는 형식화된(preformatted) text를 지정한다.
            아래에서와 같이 작성된 그대로 표시된다.
        
            var myArray = [];
            console.log(myArray.length); // 0
                            
            myArray[1000] = true;  // [ , , ... , , true ]
                            
            console.log(myArray.length); // 1001
            console.log(myArray[0]);     // undefined

q tag는 짧은 인용문을 지정한다: 이렇게 큰따옴표로 표시된다.

blockquote를 쓰면 이렇게 들여쓰기 된 긴 인용문 블록 설정이 가능하다.
