List

1. 순서 없는 목록 (Unordered List)

  • ul tag를 먼저 열고
  • 그 안에 li tag로 item을 작성한다.
  • 이후 /ul tag로 닫아준다.

2. 순서있는 목록 (Ordered List)

  1. ol tag를 먼저 열고
  2. 그 안에 li tag로 item을 작성한다.
  3. 이후 /ol tag로 닫아준다.

2-1. type attribute

  1. 대문자 로마숫자로 써 보기
  2. ol tag 안에 type 설정을 해 주면 된다.
  3. li tag 안에 value 설정해서 번호 지정도 가능하다.

2-2. start attribute

  1. ol tag 안에 start 설정을 해 주면
  2. 이렇게 시작 번호를 설정해 줄 수도 있다.
  3. 당연히 start="3"으로 세팅했다.

2-3. reversed attribute

  1. ol tag 뒤에 reversed를 넣어주니
  2. 이렇게 순서가 뒤집힌다.
  3. 그냥 ol reversed 가 전부다.
